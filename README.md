# Netflix Data Analysis Project

## Overview
This project analyzes Netflix dataset using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Done
- Data Cleaning
- Movies vs TV Shows (Pie Chart)
- Year-wise Trend (Line Chart)
- Top Countries (Bar Chart)
- Ratings Analysis
- Duration Distribution (Histogram + Poisson)
- Heatmap Visualization

## Insights
- Movies are more than TV Shows
- Content increased over years
- Certain countries dominate Netflix content
